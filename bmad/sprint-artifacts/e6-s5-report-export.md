# Story 6.5: Report Export

Status: ready-for-dev

## Story

As a desktop application user (IT leader presenting to executives),
I want to export activity charts to CSV, JSON, and PDF formats with embedded chart images and metadata,
so that I can analyze data in Excel, share reports with stakeholders, and present ROI metrics in professional formats.

## Acceptance Criteria

1. Export dropdown in Activity chart toolbar with "Export as CSV", "Export as JSON", and "Export as PDF" options
2. CSV export includes header row and data rows (Date, Prompts, Tokens)
3. JSON export is valid JSON with pretty-print indentation
4. PDF export includes: formatted header, embedded chart images (both prompts and tokens charts), summary table, metadata footer
5. PDF chart images captured via Chart.js `.toBase64Image()` method
6. PDF generated using jsPDF library (A4 portrait format)
7. Export metadata included in all formats: export timestamp, date range, granularity
8. Native save dialog allows user to choose save location and filename
9. Default filenames include date: "activity-export-2025-11-26.csv" / ".json" / ".pdf"
10. Exported CSV opens correctly in Excel/Google Sheets
11. Exported PDF displays correctly in PDF viewers (Adobe Reader, Preview, Chrome)
12. Success toast shows after export completes with file path
