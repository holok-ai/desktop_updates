Terminology
Activity Bar: the left-most vertical panel containing tp-level activities
Activity List: the vertical list of items for the selected activity located to the right of the Activity Bar
Activity Area/Main Window: The right most area where most working content is displayed

Current Behavior
The Activity Bar icons are either not present or the same color as the background.
The Activity List is not resizable.

Expected Behavior
The Activity Bar should be collapsible using a "<" and ">" to collapse and expand the activity bar. this should be the same style (color and symbol) as is used in the Moku Web.
The collapse and expand should always be a white symbol since the Activity Bar uses a dark background regardless of light or dark mode.

The Activity List should be collapsible using a "<" and ">" and follow the same design as the Activity Bar.
The Activity List should also include a grabbable re-sizer (a slim vertical rectangle or similar) that allows the user to make the Activity List as wide as desired.  
If the user sets the size using the re-sizer, the expand button should restore the Activity List to the previous size.
For example, the user re-sizes the Activity List to "350px", then clicks the Activity List Collapse ("<"). When the user clicks the expand icon (">") on the Activity List, the list width is restored to "350px"
